PROJECT TITLE: The Domino Effect
PURPOSE OF PROJECT: Final for CSCI145 
VERSION or DATE: 12/14/2015
HOW TO START THIS PROJECT: Run then Press Space Bar
AUTHORS: Keila Calderon
USER INSTRUCTIONS: Press run then space bar to begin the game. Use your arrow keys to move across the screen. Your goal is to capture the bombs that are flying out of the bomber. You will have successfully won if you have reached a score greater than 0 at the end of the timer. If you have reached a negative score you have lost the game.
